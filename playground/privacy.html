Privacy Policy

Agent Snap Chrome extension privacy policy

Agent Snap only runs when you activate it on the current tab. It is designed to help you annotate page elements, capture visual context, and export that context for your own workflow.

Last updated: March 14, 2026

What data Agent Snap processes

  • The current page URL and the DOM details needed to identify annotated elements.
  • Annotation text, comments, and attachments that you add while using the tool.
  • Screenshots you choose to capture as part of an annotation.
  • Local settings and saved annotations stored in your browser for the current page.
  • Clipboard output only when you explicitly use the copy or export actions.

How Agent Snap uses that data

  • To render the annotation toolbar and markers on the page you selected.
  • To save your annotations locally so they persist while you work.
  • To generate the Markdown snapshot and asset manifest you request.
  • To copy the generated output to your clipboard when you choose that action.

When data leaves your browser

Agent Snap does not automatically collect analytics, sell your data, or send browsing activity to ad networks.

If screenshot uploads are enabled, screenshots and attachments you capture can be uploaded to the Agent Snap upload service at https://agent-snap.conekto.eu/api/public/upload so the generated output can reference hosted asset URLs.

If you disable screenshot uploads in the tool settings, captured assets stay in the generated output as local data URLs and are not uploaded by the extension.

Chrome permissions

  • activeTab is used so Agent Snap can run on the page you explicitly choose.
  • scripting is used to inject the annotation UI into that active page.

Your choices

  • You can close the tool at any time.
  • You can clear saved annotations from the toolbar.
  • You can disable screenshot uploads in the settings before exporting content.
  • You should avoid using the extension on pages containing information you do not want included in annotations or screenshots.

Contact

For privacy questions or requests, open an issue at github.com/vampaz/agent-snap/issues .
